COMO POSSO FORMULAR UM PROBLEMA?

  • Colete os dados:

    • Reúna dados e trabalhe para construir arranjos a partir deles

  • Análise os dados:

    • Examine os dados que você coletou

  • DECOMPONHA:

    • Divida seu problema em um conjunto de problemas menores e mais simples

  • IDENTIFIQUE PADRÕES:

    • Localize dados que se repetem

  • Trabalhe a ABSTRAÇÃO:

    • Simplifique os processos selecionando características importantes do problema em questão

  • Crie ALGORITIMOS:

    • Desenvolva uma sequência de pequenas soluções (algoritimo) para o seu problema

  • Construa modelos:

    • Simule, teste, experimente, procure falhas.
