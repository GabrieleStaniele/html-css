Principais formatações

Resumo da aula:

Negrito e itálico: FORMA

Strong e Enfase: SEMÂNTICA, recomendado.

Comando "strong" para partes fortes da frase

Comando "em" para dar enfâse a parte da frase

CTRL + SHIFT + P = atalho para comandos e paragrafos. = abb

Negrito / Destaque:

Nessa frase temos um Termo em negrito usando a tag B (não tem significado, só forma - não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (semântica)

Itálico /Ênfase:

Nesta frase, temos um termo em itálico usando a tag I (não semântica)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto Marcado

Podemos criar um texto marcado usando a tag MARK.

Para marcar todos maks com a mesma cor, crie uma tag style, um seletor para mark com a cor padronizada.

Estamos criando um texto grande e um texto pequeno

Big: você não deu significado,
Small: deu sentido, pois letras menores são usadas por exemplo em contrátos.

Texto deletado

Podemos marcar um texto como excluído para indicar que pode ser lido, mas não considerado.
tag: del.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
tag: ins.

Exite também o sublinhado com a tag U(não semântica)

Texto Sobrescrito

Para inserir informações como x20 + 3.
tag: sup.

Texto subscrito

Para inserir informações como: H20.
tag: sub.
